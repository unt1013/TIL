# Python의 저장 반복 조건

## 1. 저장

### 1.1 문자와 숫자

- 담고자 하는 자료를 변수에 담기만 하면 된다. 문자와 글자의 구분은 ' '로 한다

### 1.2 Bool

- False, True로 표현한다.



## 2. 반복

### 2.1 for

- for 변수 in List : 형식으로 사용한다.

  ```python
  for number in list :
  ```



### 2.2 while

- while 조건 : 형식으로 사용한다.

  ```python
  while boolean :
  ```



## 3. 조건

### 3.1 if

- if 조건 : 형식으로 사용한다. 아래 내용은 들여쓰기로 하며, elif와 else 또한 같은 형식으로 사용

  ```python
  if boolean :
  	something
  elif boolean :
  	something
  else :
  	something
  ```
